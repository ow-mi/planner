== Project Planning for testing program

- The program has several legs (a testing sequence)
- each leg has many tests
- each test requires specific equipmnet for the duration of the test
- each test requires some FTE (full time employee) time
- the tests in each leg must follow the sequence

- Priorities
    - the user needs to supply the leg priority list 
    - some tests may have fixed dates or deadlines
    - each FTE adn equipment can only be used by one test at a time
